# Ex-05 — Grow

## Objectif
Faire en sorte qu'un seul item prenne tout l'espace restant.

## Règles
- Modifier uniquement `style.css`.
- Utiliser uniquement Flexbox.
- `styles/base.css` est intouchable.

## Contraintes
- HTML inchangé.
- Pas de CSS Grid.
- Pas de `position` pour la mise en page.

## Auto-validation
- [ ] L'item B s'étire et occupe l'espace libre.
- [ ] Les items A et C gardent leur taille.
- [ ] Le rendu reste propre en redimensionnant la fenêtre.
