# Flexbox Training

Ce dépôt propose une série d'exercices pour débuter avec Flexbox. Chaque dossier correspond à un concept précis vu en classe.

## Principe

- 1 dossier = 1 concept Flexbox.
- Ouvrir simplement `index.html` dans un navigateur.
- Le fichier à modifier est `style.css` dans chaque exercice.
- `styles/base.css` contient le style visuel commun et ne doit **jamais** être modifié.

## Mode solution

Pour afficher la solution, ajoutez la classe `.is-solved` sur `.page` dans le HTML.
**Ne pas** utiliser ce mode pendant l'exercice, jouez le jeu !

## Contenu

- 8 exercices progressifs.
- Consignes courtes et checklist d'auto-validation par exercice.
