# Ex-06 — Align-self

## Objectif
Aligner un item différemment du groupe sans toucher au HTML.

## Règles
- Modifier uniquement `style.css`.
- Utiliser uniquement Flexbox.
- `styles/base.css` est intouchable.

## Contraintes
- HTML inchangé.
- Pas de CSS Grid.
- Pas de `position` pour la mise en page.

## Auto-validation
- [ ] Les items 1, 3, 4 sont centrés verticalement.
- [ ] L'item 2 est placé plus bas que les autres.
- [ ] Je n'ai pas utilisé de marges ou de positionnement.
