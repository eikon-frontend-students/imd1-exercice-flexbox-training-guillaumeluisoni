# Ex-02 — Distribuer

## Objectif
Répartir logo, nav et action sur toute la largeur de l'arena.

## Règles
- Modifier uniquement `style.css`.
- Utiliser uniquement Flexbox.
- `styles/base.css` est intouchable.

## Contraintes
- HTML inchangé.
- Pas de CSS Grid.
- Pas de `position` pour la mise en page.

## Auto-validation
- [ ] Les blocs touchent les bords gauche et droit.
- [ ] Le bloc central reste entre les deux autres.
- [ ] Les trois blocs sont alignés verticalement.
