# Ex-08 — Synthèse

## Objectif
Construire un layout cohérent et fluide en combinant plusieurs règles Flexbox.

## Règles
- Modifier uniquement `style.css`.
- Utiliser uniquement Flexbox.
- `styles/base.css` est intouchable.

## Contraintes
- HTML inchangé.
- Pas de CSS Grid.
- Pas de `position` pour la mise en page.
- Auto-validation stricte sur le comportement responsive.

## Auto-validation
- [ ] La barre du haut est alignée et répartie sur la largeur.
- [ ] Le bloc Main s'étire et reste prioritaire.
- [ ] En petite largeur, les blocs passent à la ligne sans chevauchement.
