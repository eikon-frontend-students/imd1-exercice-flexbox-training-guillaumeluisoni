# Ex-07 — Order

## Objectif
Changer l'ordre visuel des items sans modifier le HTML.

## Règles
- Modifier uniquement `style.css`.
- Utiliser uniquement Flexbox.
- `styles/base.css` est intouchable.

## Contraintes
- HTML inchangé.
- Pas de CSS Grid.
- Pas de `position` pour la mise en page.

## Auto-validation
- [ ] L'ordre visuel est 2 - 3 - 1 - 4.
- [ ] Le HTML est intact.
- [ ] Je n'ai utilisé que la propriété `order`.
