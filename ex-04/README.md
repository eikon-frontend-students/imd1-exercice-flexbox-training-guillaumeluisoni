# Ex-04 — Wrap

## Objectif
Permettre aux items de revenir à la ligne automatiquement.

## Règles
- Modifier uniquement `style.css`.
- Utiliser uniquement Flexbox.
- `styles/base.css` est intouchable.

## Contraintes
- HTML inchangé.
- Pas de CSS Grid.
- Pas de `position` pour la mise en page.

## Auto-validation
- [ ] Les items passent à la ligne sans chevauchement.
- [ ] L'espace horizontal et vertical est régulier.
- [ ] Le rendu reste propre en redimensionnant la fenêtre.
