# Ex-01 — Centrer

## Objectif
Centrer 3 items horizontalement et verticalement dans l'arena.

## Règles
- Modifier uniquement `style.css`.
- Utiliser uniquement Flexbox.
- `styles/base.css` est intouchable.

## Contraintes
- HTML inchangé.
- Pas de CSS Grid.
- Pas de `position` pour la mise en page.

## Auto-validation
- [ ] Les items sont centrés sur l'axe horizontal.
- [ ] Les items sont centrés sur l'axe vertical.
- [ ] L'espacement entre items est régulier.
